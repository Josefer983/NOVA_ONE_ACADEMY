# 🚀 NOVA ONE Academy

NOVA ONE Academy es un proyecto educativo creado para aprender desarrollo web construyendo un Punto de Venta (POS) moderno utilizando HTML, CSS y JavaScript.

Este proyecto servirá como laboratorio para posteriormente desarrollar NOVARA ONE con Next.js.

---

## 📚 Tecnologías

- HTML5
- CSS3
- JavaScript (ES6+)
- Local Storage

---

## 🎯 Objetivos

- Aprender desarrollo Frontend.
- Crear un sistema POS profesional.
- Practicar arquitectura de proyectos.
- Aprender buenas prácticas.
- Crear componentes reutilizables.
